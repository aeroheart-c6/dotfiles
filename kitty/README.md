kitty
=====

Configurations for kitty terminal with keybindings and other customisations

## Files

All the files here are relative to:

    ~/.config/kitty

| Name           | Location               |
| -------------- | ---------------------- |
| `kitty.conf`   | `./kitty.conf`         |
| `keymaps.conf` | `./keymaps.conf`       |
| `theme.conf`   | `./current-theme.conf` |


## Theming

Use the following command to change the theme easily

    kitten themes
